## рџ’Ў Tips for Maintenance

### Adding New Content
1. Add data to appropriate file in `src/data/`
2. Component will automatically render it
3. No code changes needed

### Modifying Animations
1. Find component file
2. Locate `useEffect` with GSAP code
3. Modify animation (keep timings same!)
4. Test in browser

### Adding New Sections
1. Create component in `src/components/`
2. Add animations with `gsap.context()`
3. Add cleanup in useEffect return
4. Import and render in App.tsx
